But how does it work ?
  • Sign up to get an account
    • At the moment, there is no email verification.
  • Use the navigation links above to look at books and meetings
  • You can only see your books. This keeps things anonymous.
  • Once signed in, you can add books. At any given time, you can only have {% settings_value 'MAX_FREE_BOOKS' %} books that aren't assigned to a meeting. Don't worry, the "Add more books" link will appear / dissappear as neeeded. You can update any of your books at any time. You can delete any of your books so long as it hasn't been assigned to a meeting.
  • You can look at a list of all meetings, past and present.
    • Only special users {% if user.profile.is_special %} (that's you!) {% else %} (not you!) {% endif %} can create a new meeting. Upon creation, a meeting is automatically allocated a random unassigned book.
